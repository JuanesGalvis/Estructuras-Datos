## Curso de Estructuras de Datos en JavaScript

1. Array
    - Acceder (get)
    - Agregar (push)
    - Eliminar (pop y delete)
2. Strings
3. Hash Table
    - Insertar (set)
    - Buscar (get)
    - Mostrar todas las keys existentes (getAll)
    - Eliminar (delete)
4. Listas (Linked List)
    - Simple
        - Agregar al final
        - Agregar al inicio
        - Agregar por posición
        - Eliminar (primero, último e intermedio)
    - Doble
        - Agregar al final
5. Pilas (Stacks)
    - Apilar
    - Desapilar
    - Mostrar el último agregado
    - Mostrar el primer agregado
6. Colas (Queue)
    - Agregar al final
    - Remover el primero
    - Mostrar el primero
7. Árboles
    - Insertar datos
    - Buscar Nodo por valor
